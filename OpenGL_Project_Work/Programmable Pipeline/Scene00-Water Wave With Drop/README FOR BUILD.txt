Before running "command.bat" follow this procedure for Bullet physics library files:

1. Build Bullet Physics on visual studio installed on your machine (select 32 bit or 64 bit as per need).
2. Copy library files from "...\bullet3-master\lib\Release" to "...\Scene00-Water Wave with Drop\INCLUDE\BulletPhysicsRohan\lib".
3. Run "command.bat".
